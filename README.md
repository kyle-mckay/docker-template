# docker-template

A template repository for initializing into docker 

## Usage

```sh
git clone https://github.com/kyle-mckay/docker-template.git [Optional: /path/to/target]
```

(Optional) allow script usage

```
sudo chmod +x [path/to/]dc.sh
```

You can now use `dc.sh` with `./dc.sh` isntead of `bash dc.sh`
